# Algorithm-Visualizer

An interactive web-based tool to visualize how different algorithms work step-by-step. This project helps users understand Data Structures and Algorithms through animations.

## 🚀 Features

* 📊 Visual representation of arrays using bars
* 🔁 Step-by-step algorithm execution
* 🎛 User input & random array generation
* 🎬 Smooth animations for better understanding
* ⏱ Speed control (coming soon)
* 🎨 Color highlighting for comparisons and swaps

## 🧠 Algorithms Included

### 🔹 Sorting Algorithms

* Bubble Sort
* Selection Sort 
* Insertion Sort
* Merge Sort 
* Quick Sort 

### 🔹 Searching Algorithms

* Linear Search 
* Binary Search 

### 🔹 Graph Algorithms 

* BFS (Breadth First Search)
* DFS (Depth First Search)

## ▶️ How to Run

Follow these steps to run the project locally:

1. Clone the repository:
git clone https://github.com/your-username/Algorithm-Visualizer.git

2. Navigate into the project folder:
cd Algorithm-Visualizer

3. Open the project:
* Open `index.html` in your browser
  **OR**
* Use Live Server in VS Code (recommended)

## 📸 Preview

### 🔹 Input & Visualization

* Enter numbers (e.g., `5 3 8 1`)
* Click **Create Array**
* Bars will be displayed representing values

### 🔹 Random Array

* Click **Random Array**
* Automatically generates and visualizes data
